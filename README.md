This repository contains basic Java programs created for learning and strengthening programming fundamentals.
It is useful for beginners who are starting with Java and want to practice core concepts.

Palindrome Number
Factorial
Fibonacci series
HCF & LCM

🔤 String Programs
Count vowels and consonants
String comparison
Reverse string

🔁 Loop & Pattern Programs
Pyramid patterns

📊 Sorting & Arrays
Lexicographical sorting
Basic array operations

🛠️ Technologies Used
Java (Core Java)
VS Code 

🚀 Purpose
1.To build a strong foundation in Java
2.To prepare for coding rounds and technical interviews
3.To practice logic building

📂 Folder Structure
Each program is organized into separate files for easy understanding.

💡 How to Run
1.Clone the repository
2.Open in any Java IDE
3.Compile and run the .java file
